#打扑克双人剧烈运动视频

书房可真大，王天一寸一寸的找，翻遍了书房也没有找到保险箱。不由骂到：[别他妈被我找到，不然有你好看！]

    [主人，我知道在哪里。]小天猛的冒出一句话把这个业余盗贼吓了一跳。[你怎么会知道我在找什么，还有别叫我主人，叫我天少。]

    [确认，天少，因为你的大脑和晶片是共享的，你想什么我都会知道。我做什么你都会了解。因为本来就是一个个体。]小天回答。[那我不是没有隐私了，说不定我今天穿什么内裤他都知道，再一想，也没什么大不了的，小天本来就是一件工具，就象电脑一样，不过是高级了一点，又会对谁去说呢。问道：[那保险箱在哪儿？]

    [在第三个房间的洗手间里，用柜子包住的就是。]操，藏的这么深，让老子一阵好找。王天心中暗骂。拿起手中的雪茄盒丢进了异空间，直接去找保险箱。

    小天叫住了他，[天少，那个电脑对我很有用。]回头一看，是那种可无线上网的限量发售笔记本，笑骂道：[你倒挺识货，我怎么把它给忘了，提醒的好，正好没有电脑。]拿起笔记本，问道：[怎么给你？]

    [你丢进异空间就可以了，我会把它的功能都复制过来。]笔记本被丢进了异空间，王天直接走向了洗手间。
